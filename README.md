## 📦 部署指南

### GitHub Pages（推荐）

1. **Fork 本仓库** 或创建新仓库
2. **上传** `cherrystudio.html` 到仓库根目录
3. **Settings** → **Pages** → Source 选择 `main` 分支
4. 等待 1-2 分钟，访问 `https://你的用户名.github.io/仓库名`

### 其他平台
- Vercel / Netlify - 拖拽部署
- Cloudflare Pages - 自动构建
- 阿里云/腾讯云 OSS - 静态托管

## ⚠️ 注意事项

1. **部分平台限制** - ChatGPT、Claude 等因安全策略禁止 iframe，会显示跳转按钮
2. **登录状态** - 各平台登录状态独立，需分别登录
3. **网络环境** - 国际平台可能需要相应网络环境
